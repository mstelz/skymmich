# Raw Idea: Session Tracking and Planning

## Feature: Session Tracking and Planning

For an astrophotography platform (Skymmich), the user wants to add:

### Session Planning (future sessions)
- Plan future imaging sessions with locations and targets
- Link to equipment/equipment groups

### Session Tracking (past/active sessions)
- Track number of light frames taken per filter
- Track calibration frames (or reference which calibration to use)
- Record weather history throughout the night
- Link to location, targets, equipment used
