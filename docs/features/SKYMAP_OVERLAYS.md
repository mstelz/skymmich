# Sky Map Overlays

## Overview

Enhancements to the existing Sky Map page that add interactive overlay layers for the user's own data.

## Features

### 1. Toggle Photographed Targets

A toggle/checkbox on the sky map that shows or hides markers for all previously photographed targets. When enabled, targets the user has already imaged appear as distinct markers (e.g., different color or icon) on the map, making it easy to see what's been captured and what gaps remain.

### 2. Saved / Tagged Targets

Allow users to save or tag interesting targets (a wishlist/planning list). These saved targets appear as a separate overlay layer on the sky map, visually distinct from photographed targets. Useful for planning future imaging sessions.

- Requires some form of a "saved targets" or "target wishlist" concept (new table or flag)
- Toggle independently from the photographed overlay
- Potential filter by tag/category
