# SPEC-011: Interactive Sky Map

**Status:** Active
**Area:** sky-and-targets
**Date:** 2026-02-25

---

## Purpose

This feature provides an interactive sky atlas that visualizes where the user's plate-solved images are positioned on the celestial sphere. It helps astrophotographers see their sky coverage at a glance and discover areas they haven't imaged yet.

---

## Behavior

The sky map page embeds an Aladin Lite v3 viewer showing a full-sky survey with markers for each plate-solved image positioned by RA/Dec coordinates.

### Entry Points

- `/sky-map` — Sky map page
- `GET /api/sky-map/markers` — Fetch plate-solved images as map markers

### Key Flows

1. Page loads and fetches markers from the API
2. Only plate-solved images with valid RA and Dec values are included as markers
3. Each marker contains: image ID, title, RA, Dec, thumbnail URL, object type, constellation, and field of view
4. Markers are rendered on the Aladin Lite sky atlas at their RA/Dec positions
5. Users can pan, zoom, and interact with the atlas
6. Equipment FOV overlay shows the field of view for linked equipment configurations

---

## Patterns Used

- Aladin Lite v3 (CDS) for sky atlas rendering
- Server-side marker projection from image metadata
- Lazy data fetching (markers loaded after page mount)

---

## Business Rules

- **Plate-solved images only:** Only images with `plateSolved: true` and non-null RA/Dec appear on the map.
- **Marker data is minimal:** The API returns only the fields needed for map display, not full image records.

---

## Edge Cases & Constraints

- Images without valid RA/Dec (even if plate-solved flag is true) are filtered out
- The Aladin Lite viewer loads external sky survey tiles from CDS servers — requires internet connectivity

---

## Known Limitations

- No clustering of overlapping markers at low zoom levels
- No filter controls on the sky map (all plate-solved images are always shown)
- No click-through from a sky map marker to the image overlay
- FOV overlay depends on equipment being linked to images with telescope specifications
